高仿新浪微博客户端
-----

一款独立开发的高仿新浪微博客户端  

1. 基本做到90%的细节还原，没有任何广告，现在任意用户均可以登录微博进行体验
2. 非常轻量化的安装包，更加流畅，快速的用户体验
3. 吸纳其他第三方微博的非常棒的功能细节，比如微博图片一键分享到朋友圈，定时微博，自定义微博来源等等(仍然再开发中)
4. 支持定制化自己的欢迎界面，您可以定制属于喜欢的图片作为欢迎界面(仍然再开发中)
